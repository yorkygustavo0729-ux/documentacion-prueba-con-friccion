# Prueba con Fricción - Documentación

## Fricción Estática y Cinética

La fricción estática máxima se puede calcular utilizando la siguiente expresión matemática:

$$
F_{friccion\_estatica\_max} = \mu_s \cdot N
$$

Donde:
- $F_{friccion\_estatica\_max}$ es la fuerza de fricción estática máxima
- $\mu_s$ es el coeficiente de fricción estática
- $N$ es la fuerza normal

### Condición para el movimiento

La fuerza aplicada $F_{aplicada}$ debe superar este valor para que el objeto comience a moverse:

$$
F_{aplicada} > F_{friccion\_estatica\_max}
$$

Solo cuando se cumple esta condición, el objeto abandonará el estado de reposo.

### Fricción cinética

Cuando el objeto está en movimiento, la fuerza de fricción se reduce y se describe mediante:

$$
F_{friccion\_cinetica} = \mu_k \cdot N
$$

Donde:
- $F_{friccion\_cinetica}$ es la fuerza de fricción cinética
- $\mu_k$ es el coeficiente de fricción cinética

**Relación importante:** $\mu_k$ es generalmente menor que $\mu_s$:

$$
\mu_k < \mu_s
$$

Esta diferencia explica por qué es más difícil iniciar el movimiento que mantenerlo.
